# C# 코딩 테스트 (3) — 반복문 & 별찍기
> 대상: 조건문에 이어 반복문(`for`)과 배열을 배우는 수강생
> 사용 문법: 앞 세트의 모든 문법 + `for` 반복문 + 중첩 반복문 + (심화 문제) 배열
> 별찍기는 **반복 안에 반복이 들어가는(중첩) 구조**를 눈으로 익히기에 가장 좋은 소재입니다.
> 입력값은 한 줄에 하나씩 주어집니다.
---
## 이번 세트에서 새로 쓰는 문법
### 1. `for` 반복문 — 정해진 횟수만큼 같은 일을 반복한다
```csharp
for (int i = 1; i <= N; i++)
{
    // 이 안이 N번 반복 실행됩니다.
}
```
- `int i = 1` : 시작 값 (i를 1부터)
- `i <= N` : 반복을 계속할 조건 (i가 N 이하인 동안)
- `i++` : 한 번 돌 때마다 i를 1씩 증가
### 2. 중첩 반복문 — 반복 안에 또 반복
```csharp
for (int i = 1; i <= N; i++)      // 바깥 반복: 줄(세로)을 담당
{
    for (int j = 1; j <= i; j++)  // 안쪽 반복: 한 줄 안의 별(가로)을 담당
    {
        Console.Write("*");       // 줄바꿈 없이 별 하나만 찍기
    }
    Console.WriteLine();          // 한 줄을 다 찍었으면 줄바꿈
}
```
> 별찍기의 핵심 도구 두 가지:
> - `Console.Write("*")` : **줄을 바꾸지 않고** 그 자리에 이어서 출력합니다. 별을 옆으로 나란히 붙일 때 씁니다.
> - `Console.WriteLine()` : 아무것도 안 쓰고 괄호를 비우면 **줄바꿈만** 합니다. 한 줄을 마무리할 때 씁니다.
>
> 바깥 반복이 "몇 번째 줄인가", 안쪽 반복이 "그 줄에 별을 몇 개 찍는가"를 맡는다고 생각하면 대부분의 별찍기가 풀립니다.

# 문제 1. 별 한 줄 찍기  *(기초)*
## 문제 설명
정수 N이 주어집니다. 별(`*`)을 한 줄에 N개 출력하세요.
예를 들어 N이 5이면 `*****` 를 출력합니다.
## 입력 형식
- 첫째 줄에 정수 N이 주어집니다.
## 출력 형식
- 별 N개를 한 줄에 출력합니다.
## 예제 입력
```
5
```
## 예제 출력
```
*****
```
## 힌트
- 별을 옆으로 나란히 붙여야 하므로, 반복 안에서는 줄을 바꾸지 않는 `Console.Write("*")`를 씁니다.
- 뼈대는 이렇게 생겼습니다.
```csharp
for (int i = 1; i <= N; i++)
{
    Console.Write("*");   // 줄바꿈 없이 별 하나 찍기
}
```
- N번 도는 동안 별이 한 자리씩 옆으로 쌓여 한 줄이 완성됩니다. 맨 마지막에 `Console.WriteLine();`을 한 번 넣어 줄을 마무리하면 깔끔합니다.
### 템플릿 코드
```csharp
using System;
public class Program
{
    public static void Main()
    {
        // 정수 N을 입력받으세요.
        // 반복문으로 별을 N개 이어서 출력하세요.
    }
}
```
### 테스트 케이스
| # | 입력 | 출력 |
|---|------|------|
| 1 | `5` | `*****` |
| 2 | `1` | `*` |
| 3 | `3` | `***` |

# 문제 2. 별 사각형  *(기초)*
## 문제 설명
정수 N이 주어집니다. 가로 N개, 세로 N개의 별로 이루어진 정사각형을 출력하세요.
예를 들어 N이 3이면 별이 3개씩, 3줄로 출력됩니다.
## 입력 형식
- 첫째 줄에 정수 N이 주어집니다.
## 출력 형식
- 한 줄에 별 N개씩, 총 N줄을 출력합니다.
## 예제 입력
```
3
```
## 예제 출력
```
***
***
***
```
## 힌트
- 문제 1에서 만든 "별 한 줄"을 **여러 줄 반복**하면 사각형이 됩니다. 그래서 반복문이 두 겹(중첩)이 됩니다.
- 바깥 반복은 줄(세로)을, 안쪽 반복은 한 줄 안의 별(가로)을 담당합니다.
- 뼈대는 이렇게 생겼습니다.
```csharp
for (int i = 1; i <= N; i++)       // 세로: N줄
{
    for (int j = 1; j <= N; j++)   // 가로: 한 줄에 별 N개
    {
        Console.Write("*");
    }
    Console.WriteLine();           // 한 줄을 다 찍으면 줄바꿈
}
```
- 사각형은 모든 줄의 별 개수가 똑같아서, 안쪽 반복의 조건이 항상 `j <= N`으로 고정입니다. 다음 문제에서는 이 부분이 줄마다 바뀝니다.
### 템플릿 코드
```csharp
using System;
public class Program
{
    public static void Main()
    {
        // 정수 N을 입력받으세요.
        // 바깥 반복문으로 N줄을 반복하세요.
        //   안쪽 반복문으로 한 줄에 별 N개를 찍으세요.
        //   한 줄이 끝나면 줄바꿈하세요.
    }
}
```
### 테스트 케이스
| # | 입력 | 출력 |
|---|------|------|
| 1 | `3` | `***`<br>`***`<br>`***` |
| 2 | `1` | `*` |
| 3 | `2` | `**`<br>`**` |

# 문제 3. 왼쪽 정렬 삼각형  *(응용)*
## 문제 설명
정수 N이 주어집니다. 별의 개수가 한 줄씩 늘어나는 삼각형을 출력하세요. 첫 줄에는 별 1개, 둘째 줄에는 2개, … N번째 줄에는 N개를 출력합니다.
## 입력 형식
- 첫째 줄에 정수 N이 주어집니다.
## 출력 형식
- i번째 줄에 별 i개를 출력합니다. (총 N줄)
## 예제 입력
```
4
```
## 예제 출력
```
*
**
***
****
```
## 힌트
- 사각형(문제 2)과 거의 같은데, **딱 한 곳만** 다릅니다. 이 한 곳이 응용의 핵심이에요.
- 사각형에서는 안쪽 반복이 항상 `j <= N`이었지만, 삼각형은 줄마다 별 개수가 달라집니다. **몇 번째 줄이냐(i)에 따라 별 개수가 정해지죠.**
- 잘 보면, i번째 줄에는 별이 정확히 i개입니다. 첫째 줄(i=1)엔 1개, 셋째 줄(i=3)엔 3개. 그러니 안쪽 반복의 조건을 `j <= N`이 아니라 `j <= i`로 바꾸면 됩니다.
- 뼈대는 이렇게 생겼습니다.
```csharp
for (int i = 1; i <= N; i++)       // i번째 줄
{
    for (int j = 1; j <= i; j++)   // 별을 i개 (N이 아니라 i!)
    {
        Console.Write("*");
    }
    Console.WriteLine();
}
```
- 바깥 반복의 값 `i`를 안쪽 반복의 조건에 끌어다 쓰는 것 — 이게 중첩 반복문이 진짜 힘을 내는 지점입니다. 두 반복이 서로 대화하기 시작하는 셈이지요.
### 템플릿 코드
```csharp
using System;
public class Program
{
    public static void Main()
    {
        // 정수 N을 입력받으세요.
        // i번째 줄에 별을 i개씩 찍는 삼각형을 출력하세요.
        //   (안쪽 반복의 개수가 바깥 반복의 i에 따라 달라지는 것이 핵심)
    }
}
```
### 테스트 케이스
| # | 입력 | 출력 |
|---|------|------|
| 1 | `4` | `*`<br>`**`<br>`***`<br>`****` |
| 2 | `1` | `*` |
| 3 | `3` | `*`<br>`**`<br>`***` |

# 문제 4. 오른쪽 정렬 삼각형  *(응용)*
## 문제 설명
정수 N이 주어집니다. 이번에는 삼각형이 **오른쪽으로 정렬된** 모양입니다. 각 줄의 별 앞에 공백을 채워, 별들이 오른쪽 변에 나란히 맞도록 출력하세요.
## 입력 형식
- 첫째 줄에 정수 N이 주어집니다.
## 출력 형식
- i번째 줄에는 먼저 공백을 (N − i)개 찍은 뒤, 별을 i개 찍습니다. (총 N줄)
## 예제 입력
```
4
```
## 예제 출력
```
   *
  **
 ***
****
```
## 힌트
- 문제 3(왼쪽 삼각형)에 **공백 찍기 반복이 하나 더** 붙는 구조입니다. 즉 한 줄이 "공백 몇 개 + 별 몇 개"로 이루어집니다.
- 규칙을 눈으로 세어 봅시다. N이 4일 때:
| 줄 번호 i | 공백 개수 | 별 개수 |
|-----------|-----------|---------|
| 1 | 3 | 1 |
| 2 | 2 | 2 |
| 3 | 1 | 3 |
| 4 | 0 | 4 |
- 별 개수는 `i`개(문제 3과 같음), 공백 개수는 `N - i`개입니다. 두 수를 더하면 항상 N이 되는 것도 확인해 보세요. 오른쪽 변이 가지런히 맞는 이유가 여기 있습니다.
- 그래서 한 줄을 만들 때 반복이 두 번 필요합니다. 먼저 공백을, 그다음 별을 찍습니다. 공백은 별 대신 띄어쓰기 한 칸 `" "`을 찍으면 됩니다.
- 뼈대는 이렇게 생겼습니다.
```csharp
for (int i = 1; i <= N; i++)
{
    for (int j = 1; j <= N - i; j++)  // 공백 (N - i)개 먼저
    {
        Console.Write(" ");
    }
    for (int j = 1; j <= i; j++)      // 그다음 별 i개
    {
        Console.Write("*");
    }
    Console.WriteLine();
}
```
### 템플릿 코드
```csharp
using System;
public class Program
{
    public static void Main()
    {
        // 정수 N을 입력받으세요.
        // 각 줄마다:
        //   1) 공백을 (N - i)개 먼저 찍고
        //   2) 별을 i개 찍은 뒤
        //   3) 줄바꿈하세요.
    }
}
```
### 테스트 케이스
| # | 입력 | 출력 |
|---|------|------|
| 1 | `4` | `   *`<br>`  **`<br>` ***`<br>`****` |
| 2 | `1` | `*` |
| 3 | `3` | `  *`<br>` **`<br>`***` |
> 참고: 공백도 출력의 일부입니다. 눈에 보이지 않지만 별 앞의 빈칸 개수가 정확해야 정답으로 인정됩니다.

# 문제 5. 속이 빈 마름모  *(심화 · 초고난도)*
## 문제 설명
홀수인 정수 N이 주어집니다. 별로 그린 **속이 빈 마름모(다이아몬드)**를 출력하세요. 테두리에만 별이 있고, 안쪽은 공백으로 비어 있습니다.
예를 들어 N이 5이면 아래와 같은 모양입니다. (윗변이 위로 뾰족해졌다가, 가운데에서 가장 넓어지고, 다시 아래로 좁아집니다.)
## 입력 형식
- 첫째 줄에 홀수 정수 N이 주어집니다.
## 출력 형식
- 총 N줄로 이루어진, 속이 빈 마름모를 출력합니다.
## 예제 입력
```
5
```
## 예제 출력
```
  *
 * *
*   *
 * *
  *
```
## 힌트
이 문제는 오늘의 정점입니다. 지금까지 배운 것들을 **한자리에 모아** 풀어야 하니, 천천히 세 걸음으로 쪼개 봅시다.
**걸음 1. 배열로 "각 줄의 별 위치"를 미리 계산해 두기**
마름모는 위로 갈수록 넓어지다가 다시 좁아지므로, 줄마다 "별이 어디에 있는지"가 규칙적으로 변합니다. 각 줄에서 별은 딱 두 곳(왼쪽 끝, 오른쪽 끝)에 있고, 맨 위와 맨 아래 줄만 한 곳(가운데)에 있습니다.
각 줄의 별 위치를 담아 둘 **배열**을 하나 만들면 생각이 편해집니다. 배열은 값을 여러 칸에 나란히 담아 두는 상자예요.
```csharp
// 예: N이 5일 때, 각 줄의 "왼쪽 별까지의 공백 수"를 담는 배열
int[] space = new int[N];   // 칸이 N개인 정수 배열
```
**걸음 2. 규칙을 표로 정리하기**
N이 5일 때, 가운데 줄을 기준으로 위아래가 대칭입니다. 가운데 줄의 번호는 `N / 2`(정수 나눗셈이라 5/2=2, 즉 0부터 세면 3번째 줄)입니다. 각 줄을 "왼쪽 공백 수"와 "두 별 사이 간격"으로 나눠 표로 적어 보세요.
| 줄 i (0부터) | 왼쪽 공백 | 별 사이 간격 | 모양 |
|--------------|-----------|--------------|------|
| 0 | 2 | (별 1개) | `  *` |
| 1 | 1 | 1 | ` * *` |
| 2 | 0 | 3 | `*   *` |
| 3 | 1 | 1 | ` * *` |
| 4 | 2 | (별 1개) | `  *` |
- 가운데(i=2)로 갈수록 왼쪽 공백은 줄고, 두 별 사이 간격은 늘어납니다. 가운데를 지나면 정확히 거꾸로 돌아옵니다.
- "가운데까지의 거리"를 구하면 규칙이 하나로 잡힙니다. 가운데 줄 번호를 `mid = N / 2`라 할 때, i번째 줄이 가운데에서 얼마나 떨어졌는지는 절댓값 개념(문제 세트 2의 그 절댓값!)으로 구할 수 있습니다. 떨어진 거리를 `d`라 하면, **왼쪽 공백 = d**, **두 별 사이 간격 = (가운데 폭) − 2×d** 형태로 정리됩니다.
**걸음 3. 한 줄을 조립하기**
한 줄은 이렇게 만들어집니다.
- 먼저 왼쪽 공백을 `d`개 찍는다.
- 별 하나를 찍는다.
- 그 줄이 맨 위/맨 아래(가운데에서 가장 먼 줄)라면 별이 하나뿐이므로 여기서 줄을 마친다.
- 그렇지 않다면, 두 별 사이의 공백을 찍고, 오른쪽 별 하나를 더 찍는다.
즉 한 줄 안에서도 조건문(`if`)으로 "별이 한 개인 줄인지, 두 개인 줄인지"를 갈라 줘야 합니다.
**정리하면 이 문제는:**
- 배열(각 줄 정보 담기) +
- 바깥 반복(줄) +
- 안쪽 반복(공백·별 찍기) +
- 조건문(별 1개 줄 vs 2개 줄) +
- 절댓값(가운데에서의 거리)
… 지금까지 배운 모든 도구가 한 번에 손을 맞잡는, 오늘의 졸업 시험 같은 문제입니다. 표를 먼저 손으로 그려 규칙을 눈으로 확인한 뒤 코드로 옮기면 길이 보일 거예요.
### 템플릿 코드
```csharp
using System;
public class Program
{
    public static void Main()
    {
        // 홀수 정수 N을 입력받으세요.
        // (선택) 각 줄의 정보를 배열에 미리 담아 두면 생각이 편해집니다.
        // 각 줄마다:
        //   1) 가운데에서 이 줄이 얼마나 떨어졌는지(거리 d)를 구하고
        //   2) 왼쪽 공백을 d개 찍은 뒤
        //   3) 별을 찍되, 별이 한 개인 줄인지 두 개인 줄인지 조건으로 나누어
        //      두 개인 줄이면 두 별 사이 공백까지 채워 출력하세요.
        //   4) 줄바꿈하세요.
    }
}
```
### 테스트 케이스
> 아래 출력에서 `·`은 실제로는 **공백**입니다. 보기 편하도록 표시만 한 것이니, 코드에서는 띄어쓰기 한 칸 `" "`을 찍으면 됩니다.
| # | 입력 | 출력 (`·` = 공백) |
|---|------|-------------------|
| 1 | `5` | `··*`<br>`·*·*`<br>`*···*`<br>`·*·*`<br>`··*` |
